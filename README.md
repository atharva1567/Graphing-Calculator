# Web-Based Interactive Arithmetic Calculator

A responsive, browser‑based graphing calculator built with HTML, CSS, and JavaScript.
It supports real‑time mathematical expression parsing and renders smooth, accurate graphs using the HTML5 Canvas API.

## Features
- Plot any function of x (e.g., sin(x), x^2, log(abs(x)), exp(x/3))
- Supports common math operations and constants: sin, cos, tan, log, sqrt, pi, e, exponentiation (^)
- Adjustable graphing window (X min/max, Y min/max)
- High‑resolution, pixel‑accurate graph rendering
- Automatic grid and axis drawing
- Live coordinate tracking on mouse hover
- Clean, responsive UI with real‑time error handling

## Technologies
- HTML
- CSS
- JavaScript (Canvas API)
